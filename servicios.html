Servicios

Existen muchos software de diseño 3D y la gran mayoría son compatibles con los formatos que se necesitan para exportar e imprimir en 3D. Pero según el objetivo de cada diseño, la elección debería ser diferente. No es lo mismo diseñar para hacer una pieza por ocio, para lo cual se puede usar un programa más elemental, que hacer un prototipo de un motor para automoción, en donde se necesita un software potente y preciso para hacer un trabajo profesional.

Nos encanta diseñar tanto piezas para ingeniería y arquitectura, como para merchandising, maquetas y regalos empresariales.
